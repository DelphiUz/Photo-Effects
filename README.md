**How to share images from Delphi FireMonkey Application?**

    procedure TFormMain.ShowShareSheetAction1BeforeExecute(Sender:TObject);
    begin
      ShowShareSheetAction1.Bitmap.Assign(ImageMain.MakeScreenshot);
    end;
